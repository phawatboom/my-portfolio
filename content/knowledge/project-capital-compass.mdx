## Project Overview

Capital Compass is a tool designed to help retail investors understand their portfolio risk and exposure.

## Tech Stack

- **Frontend**: Next.js, Tailwind CSS, Recharts
- **Backend**: Python, FastAPI, Pandas
- **Data**: Financial Modeling Prep API

## Key Learnings

- Handling large time-series datasets efficiently.
- Building interactive financial visualizations.
